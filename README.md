# 📝 Todo App (Full Stack Web Application)

A simple and responsive **Todo Application** built using **HTML, CSS, JavaScript, and Node.js**.  
This app helps users manage daily tasks with features like authentication, adding, editing, and deleting tasks.

---

## 🚀 Features

- 🔐 User Registration & Login System  
- ➕ Add New Tasks  
- ✏️ Edit Tasks  
- ❌ Delete Tasks  
- 📋 View All Tasks  
- 🚪 Logout Option  
- 🎨 Clean & Responsive UI  

---

## 🛠️ Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

### Backend
- Node.js  
- Express.js  

### Database
- Local Storage / MongoDB *(based on implementation)*
